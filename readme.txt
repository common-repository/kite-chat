With Kite Chat, you can chat with your customers via a widget on your business website.

== Description ==

Benefits of Kite Chat: 
- Chat and send file straight from website
- Know where your visitors come from
- Engage your customers with friendly onboard conversation
- Prompt visitors to chat with an automatic trigger
- Archive all conversations for better management

Setting up Kite Chat is easy
**STEP 1**: Enable Kite Chat for a group
**STEP 2**: Install plugin into your wordpress site
**STEP 3**: Go to setting and paste your kite AppId here and save

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Kite Chat screen to configure the plugin
1. At admin configurate page, paste your kite appId and save
